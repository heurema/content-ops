# Platform Limits Reference

## dev.to

| Field | Limit |
|-------|-------|
| Title | 128 chars max |
| Description | 160 chars max |
| Tags | Max 4, lowercase, hyphens only, no spaces |
| Body | Markdown supported, no raw HTML iframes |
| Cover image | Recommended 1000x420px |

## Hashnode

| Field | Limit |
|-------|-------|
| Title | 100 chars max |
| Description (subtitle) | 250 chars max |
| Tags | Max 5, {name, slug} format |
| Body | Markdown + MDX |

## Twitter/X

| Field | Limit |
|-------|-------|
| Single tweet | 280 chars |
| Thread length | No hard limit; 6-8 tweets recommended |
| Free tier | 500 post writes/month |
| Links | Count as ~23 chars regardless of actual length |

## Hacker News

| Field | Limit |
|-------|-------|
| Title | 80 chars max (UI), no clickbait |
| Submission | Human-in-loop, no API |
| Duplicates | Check Algolia before submitting |

## Reddit (deferred — v2)

API access requires pre-approved credentials (post-Nov 2025 policy).
