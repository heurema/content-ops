# Research Credits

## Structural Reference

`coreyhaines31/marketingskills` (MIT License, github.com/coreyhaines31/marketingskills) was used as a structural reference for this plugin — specifically to understand what topics a content skill set should cover.

No text was copied. All skill content in content-ops was written from scratch with a developer-centric voice and artifact-driven workflows. The marketingskills taxonomy (content-strategy, social-content, email-sequence, copy-editing, launch-strategy) informed the selection of MVP skills.
